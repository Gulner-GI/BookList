# Simple TODO API

Этот проект - это элементарный REST API для управления списком задач. Реализован на Golang с использованием фреймворка Gin.
Стек технологий: Golang, gin, JSON over HTTP...

Примеры API запросов:

 GET
 DELETE
 ...
